# EcoShare

EcoShare is a collaborative trip-planning and expense-splitting platform that encourages eco-friendly travel decisions.

## Features

- **Trip Planner**: Organize your trips, manage destinations and dates.
- **Expense Tracker**: Split costs fairly with friends and track who owes what.
- **Sustainability Insights**: Visualize your carbon footprint and earn rewards for eco-friendly choices.

## Getting Started

1.  **Install Dependencies**:
    ```bash
    npm install
    ```

2.  **Run Development Server**:
    ```bash
    npm run dev
    ```

3.  **Open in Browser**:
    Navigate to `http://localhost:5173`.

## Tech Stack

- **Frontend**: React + Vite
- **Styling**: Vanilla CSS (Eco-friendly Design System)
- **Icons**: Emoji / CSS Shapes (for lightweight performance)

## Project Structure

- `src/landing`: Landing page components.
- `src/dashboard`: Main dashboard layout.
- `src/trips`: Trip planner feature.
- `src/expenses`: Expense tracking feature.
- `src/sustainability`: Impact visualization feature.
